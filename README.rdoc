= Accounts Client

== Usage

=== For a Rails app

In Gemfile

    gem 'accounts_client'

And you should define:

    Accounts::HYDRA
    Accounts::API_KEY

=== For a non-Rails app

In Gemfile

    gem 'accounts_client'
